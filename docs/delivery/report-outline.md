# Report Outline

The individual report should be at least 3,000 words.

## Suggested Sections

1. Title Page
2. Introduction and Background of the Study
3. Statement of the Problem
4. Objectives of the Study
5. Scope of the Study
6. Literature Review
7. Methodology
8. Hardware and Software Tools
9. Data Sources
10. Database Design
11. System Implementation
12. Results and Demonstration
13. Study Limitations
14. Individual Contribution
15. Conclusion
16. References
17. Appendix

## Appendix Ideas

- ER diagram
- `CREATE TABLE` statements
- Sample seed data
- Transcript view
- GPA function
- Trigger examples
- Screenshots from the application
