# Admin Feature Spec

## Purpose

Admin manages the academic setup data and can inspect system summaries.

## Core Screens

- Dashboard
- Departments
- Students
- Lecturers
- Courses
- Academic sessions
- Semesters
- Course offerings
- Result oversight

## Required Data Operations

- Create and update departments.
- Create and update students.
- Create and update lecturers.
- Create courses.
- Create academic sessions and semesters.
- Offer courses in a semester.
- Assign lecturers to course offerings.
