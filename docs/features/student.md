# Student Feature Spec

## Purpose

Student users register courses and view their academic records.

## Core Screens

- Student dashboard
- Course registration
- My registered courses
- Results
- Transcript

## Required Data Operations

- Read available course offerings.
- Register for a course offering.
- Drop or cancel registration if allowed.
- View approved registrations.
- View semester results.
- View transcript.
